# CYBOP-Benchmark

Simple benchmark to compare CYBOL with other programming languages.

## **Pseudocode**
-----------------

```
loop from 0 to x:
  var test;
  test = Addition
  test = Substraction
  test = Division
  test = Multiplication
  Absolute value of test
  innerLoop();
end loop
  
innerLoop():
  outerResult = 0
  loop from 0 to y:
    i++;
    if(i % 2 == 0):
      innerResultCounter++
  
  outerResult += innerResultCounter
end loop
```

## **How to run**
-----------------

- navigate to CYBOB-Benchmark directory
- run ./runner.sh loops
- press start button of the GUI
- Wait for the results

_!Results are average values of all loops_
